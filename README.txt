JUNIPER & STONE — GOODFORM CONCEPT WEBSITE

A fictional, non-commercial garden design and landscaping website created as a Goodform portfolio project.

HOW TO PUBLISH
1. Create a new public GitHub repository, for example: juniper-stone-concept
2. Upload all files and the assets folder to the repository root.
3. Open Settings > Pages.
4. Select Deploy from a branch, main branch, root folder.
5. The site will be available at:
   https://YOUR-USERNAME.github.io/juniper-stone-concept/

IMPORTANT
- The page includes a visible concept-project notice.
- The website uses noindex/nofollow so it should not compete in search results as a real local business.
- The contact form is a visual demonstration and does not send messages.
- Replace or remove the concept notice only if adapting the project for a genuine client.
